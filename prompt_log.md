### Entry 1: Presentation Generation
* **Tool Used:** ChatGPT & Gamma AI
* **Goal:** Generate a professional, 8-slide presentation deck tailored to the hackathon's evaluation criteria.
* **Initial Prompt:** "Now give me a prompt I can write in gamma ai to generate a ppt which I can present it as final submission."
* **Refined Prompt (Used in Gamma):** "Create a high-impact, professional hackathon presentation for a project titled 'Smart File Organizer'. The core problem is the inefficiency of digital clutter and manually organizing computer folders. Our solution is a Python-based automation tool using the 'os' and 'shutil' modules, developed through 'Vibe Coding' (leveraging AI like ChatGPT and Copilot for rapid prototyping). Structure the deck into: Problem Statement, Solution Architecture (Python + AI), Efficiency Metrics, Prompt Engineering Process, and Future Roadmap. Style: Modern Dark Theme, minimalist, high-contrast with neon green accents."
* **Result:** Successfully generated a high-quality, structured slide deck that directly addressed the 20% "Prompt Engineering" and 20% "Working Prototype" evaluation criteria.

### Entry 2: Core Logic Generation
* **Tool Used:** ChatGPT
* **Goal:** Generate the initial Python script to traverse a directory and move files based on their extensions.
* **Prompt Used:** "Write a Python script using the os and shutil libraries that scans a target folder and moves files into subfolders (like Images, Documents, Code) based on their file extensions. Add a fallback condition for uncategorized files."
* **Result:** Generated the foundational `organizer.py` script. The logic successfully created directories and prevented file overwrites.
