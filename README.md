# 📂 Smart File Organizer

A Python-based, AI-assisted desktop automation tool built for the Tech-AI-Thon. This application scans cluttered local directories and automatically categorizes files into structured folders (Documents, Images, Code, etc.) using rule-based logic and AI categorization.

## 🚀 The Tech Stack
* **Core Logic:** Python (`os`, `shutil`)
* **GUI Framework:** PyQt6 *(in development)*
* **AI Integration:** OpenAI API *(in development)*
* **Methodology:** Built using "Vibe Coding" with ChatGPT and GitHub Copilot.

## 🧠 Evaluation Criteria
* **Working Prototype:** Check `organizer.py` for the core file-moving engine.
* **Prompt Engineering:** View our detailed [Prompt Log](prompt_log.md) to see how we iteratively built this project using AI. 

## 🛠️ How to Run
1. Clone this repository.
2. Ensure you have Python installed.
3. Run `python organizer.py`.
4. Enter the path of a messy test folder when prompted.
